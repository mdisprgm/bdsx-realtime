# bdsx-realtime
This is a bdsx plugin to set the game(minecraft) time the same as the current time.

### How to use
- Step 1. Download this code.
- Step 2. Unzip it.
- Step 3. Correct the latitude and longitude in config.json.
- Step 4. Apply it to your bdsx.
